    <p align="center">

## User Stories

- User Registers with username & password
- Authenticate Users
- User can Create, Read, Update & Delete Ideas
- Users can upvote/downvote ideas
- Users can bookmark/save ideas
- Users can comment on ideas

## Technology Stack

- Rest API : NestJS
- Database : MySQL


## Authentication 

- JWT Authentication


## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```
